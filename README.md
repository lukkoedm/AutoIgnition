# AutoIgnite

Automatically activates furnaces and campfires when fuel and input items are placed inside, so you don't have to manually press the "Turn on" button.

## Installation

1. Navigate to Hytale Mods folder:  
   `C:\Users\YourUsername\AppData\Roaming\Hytale\UserData\Mods\`
2. Copy the plugin JAR into the `Mods/` folder.
3. Launch Hytale.
4. Enable the mod in the world settings.
